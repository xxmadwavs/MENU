⚙️ Administración

Asigna recetas a cada comida. Solo aparecen recetas del tipo correspondiente. Pulsa "Guardar menú" al terminar.

📋 Cómo funciona

1. Edita recetas y menú aquí.
2. Pulsa "Generar JSON" para exportar.
3. Copia el contenido y guárdalo como data.json en tu repositorio de GitHub Pages.
4. El index.html cargará automáticamente el nuevo JSON al abrirse.
