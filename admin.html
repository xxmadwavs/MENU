📋 Cómo funciona con GitHub Pages

1. Edita tus recetas, ingredientes y menú.
2. Pulsa "Descargar data.json".
3. Sube ese archivo a tu repositorio de GitHub (misma carpeta que index.html).
4. Tu web cargará los datos automáticamente.
